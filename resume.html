技术

  1. HTML5 : 优秀的标签语言,更好的语义特性,设备兼容性,优秀的本地存储提高性能,Canvas矢量绘图功能给予更多可能.
  2. CSS3 : 出色的页面样式,支持圆角属性,投影效果,rgba()颜色,变形,转换和动画.
  3. Javascript : 弱类型脚本语言,由ECMAScript,BOM和DOM组成.使网页和用户之间形成一种实时的,动态的,交互的关系.
  4. DOM : 以面向对象方式描述的文档对像模型,通过js和文档树(节点层次)能够完成对页面的重构.
  5. jQuery : 使用了CSS选择器,跨浏览器的JavaScript库,具有动态特效,优秀的AJAX,链式操作和强大的插件扩展.
  6. AJAX : 通过XmlHttpRequest使得前台可以异步请求后台数据,完成界面局部刷新而不影响其他功能,具有更迅捷的响应速度.
  7. bootstrap : 由动态CSS语言Less写成的CSS/HTML框架,通过内置的丰富web组件样式表和glphyicon图标以及jQuery的操作,使其可以进行快速开发和强大的样式布局.
